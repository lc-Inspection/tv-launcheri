Kurulum Rehberi
1

Google Sheet'i hazırlayın

Sheet ID zaten Code.gs içinde tanımlı. Sadece Apps Script'i açmanız yeterli: Sheet → Uzantılar → Apps Script

2

Code.gs'i yapıştırın ve initSheets çalıştırın

  • Eski kodu silin, yeni Code.gs'i yapıştırın
  • Fonksiyon: initSheets → ▶ Çalıştır (Google izni isteyecek)
3

Yeniden dağıtın

  • Dağıt → Dağıtımları yönet → Kalem ikonu → Sürüm: Yeni sürüm → Dağıt
  • Ya da: Dağıt → Yeni dağıtım → Web uygulaması → Erişim: Herkes
4

Giriş ekranında URL'i girin

Çıkış yapıp giriş ekranına dönün. Şifre + Apps Script URL'ini girin.

5

Samsung TV'ye URL yapıştırın

Ekran kartındaki URL Kopyala butonuna basın. TV → URL Başlatıcı → Web Uygulamasını Yükleme alanına yapıştırın.
